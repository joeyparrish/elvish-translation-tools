# Sindarin Consonant Mutations

Source: https://www.elvish.org/gwaith/sindarin_phonetics.htm (Gwaith-i-Phethdain)

There are three primary mutations in mature Sindarin (soft / lenition, nasal,
hard / stop), plus two more speculative ones (mixed, liquid) proposed by David
Salo and used in some Neo-Sindarin reconstructions but rejected by some
scholars. Treat mixed/liquid mutations as optional and flag them.

## Soft mutation (lenition)

Triggered by: prepositions `ab, adel, am, be, di, go-, na, nu, trî, u-`;
compounds; adjectives following nouns; direct objects of verbs;
negative commands with `avo`.

| Basic   | Mutated |
|---------|---------|
| b, bl, br | v, vl, vr |
| c, cl, cr | g, gl, gr |
| d, dr     | dh, dhr |
| f         | f (no change) |
| g, gl, gr, gw | (silent), 'l, 'r, 'w |
| h         | ch |
| hw        | chw |
| p, pr     | b, br |
| s         | h |
| t, tr     | d, dr |
| th        | th (no change) |
| m         | v |
| n, l, r, lh, rh | unchanged in most analyses |

## Nasal mutation

Triggered by: plural article `in`, preposition/prefix `an` "for/to",
preposition `dan` "against".

| Basic | Mutated |
|-------|---------|
| b     | m |
| d     | n |
| g     | ng |
| p     | ph |
| t     | th |
| c     | ch |
| s     | s (often) |
| h     | ch |

Note: `an` + initial vowel often surfaces as `na`/`'n`.

## Hard mutation (stop mutation)

Triggered by: prepositions `ed` "out of", `ned` "in", `o` / `od` "from".

| Basic | Mutated |
|-------|---------|
| p     | ph |
| t     | th |
| c     | ch |
| b     | p |
| d     | t |
| g     | c |

## Mixed mutation (speculative)

Triggered (per Salo) by: article `en` (singular genitive), and prepositional
contractions `ben, erin, nan, uin, 'nin`. Mostly equivalent to soft but with
some differences (e.g., `b → b`, not `v`).

## Liquid mutation (speculative)

Triggered (per Salo) by certain prepositions like `or` "above". Mostly
produces aspirated forms similar to hard mutation.

## Practical rules of thumb

- Adjectives follow their nouns and undergo soft mutation: `aran pant`
  → `aran bant`? (No: `p → b` after lenition, so `pant` → `bant` after a noun
  ending in a vowel or sonorant). Always check.
- Direct objects of verbs are lenited.
- The definite article `i` (singular) causes soft mutation;
  `in` (plural) causes nasal mutation.
- After prepositions, apply the mutation listed for that preposition.
- Compounds typically lenite the second element.
